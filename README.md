## 建博初衷

1. 有些东西看能看懂，会写。但是给能给别人讲明白又是另外一回事了。通过写文章，能更加深入了解各个知识点。
2. 经常查问题的时候，看到有些人写的文章，满篇名词，晦涩难懂，看了简直要抓狂了，当时的感觉就是，给自己一巴掌，怎么这么笨？给博主一巴掌，能讲人话吗！所以我致力于写最通俗易懂的博客。
